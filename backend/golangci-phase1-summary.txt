## New Violations by Linter (Phase 1 Additions)

### 1. mnd (Magic Number Detection) - 719 violations
   - Detects hardcoded numeric literals
   - Target: Extract to named constants
   - Example: timeout := 30 → timeout := defaultTimeoutSeconds

### 2. perfsprint (Performance Optimization) - 565 violations
   - Detects inefficient string formatting
   - Target: Use faster alternatives
   - Example: fmt.Sprintf("%d", n) → strconv.Itoa(n)

### 3. funlen (Function Length Limits) - 203 violations
   - Lines: 80 max
   - Statements: 50 max
   - Target: Split long functions into smaller units

### 4. dupl (Duplicate Code Detection) - 63 violations
   - Detects copy-pasted code blocks
   - Target: Extract to shared functions

### 5. goconst (Repeated Strings → Constants) - 60 violations
   - Detects repeated string literals
   - Target: Extract to named constants
   - Example: "error" repeated 5+ times → const errPrefix = "error"

### 6. nolintlint (Validate //nolint Usage) - 23 violations
   - Ensures //nolint directives are valid
   - Target: Fix or remove invalid suppressions

### 7. gochecknoglobals (No Global Variables) - 17 violations
   - Detects global variable declarations
   - Target: Refactor to dependency injection or config structs
