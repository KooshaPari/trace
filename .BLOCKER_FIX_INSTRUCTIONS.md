## FIX STEPS

### Step 1: Update tsconfig.json (1 minute)

**File:** `frontend/apps/web/tsconfig.json`

**Change Line 38 FROM:**
```json
    "types": ["vitest/globals"]
```

**Change Line 38 TO:**
```json
    "types": ["vitest/globals", "node", "@testing-library/jest-dom", "vite/client"]
```

**Why:** This adds missing type definitions for:
- `node`: Provides `process`, `require`, `NodeJS.Timeout` types
- `@testing-library/jest-dom`: Testing library types
