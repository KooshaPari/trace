# Config Files

Configure the TraceRTM CLI using configuration files for persistent settings.

## Config File Location

Configuration files are stored in:

- **macOS/Linux**: `~/.config/tracertm/config.json`
- **Windows**: `%APPDATA%\tracertm\config.json`

## Config File Format

```json
{
  "api-url": "https://api.tracertm.dev",
  "api-key": "rtm_live_abc123...",
  "default-project": "proj_123",
  "output-format": "json",
  "timeout": 30,
  "verbose": false
}
```

## Editing Config

### Using CLI

```bash
# Set values
trace config set api-url https://api.tracertm.dev
trace config set api-key rtm_live_abc123...

# View config
trace config show
```

### Manual Editing

Edit the config file directly:

```bash
# macOS/Linux
nano ~/.config/tracertm/config.json

# Windows
notepad %APPDATA%\tracertm\config.json
```

## Config Options

| Option | Type | Description | Default |
|--------|------|-------------|---------|
| `api-url` | string | API endpoint | `https://api.tracertm.dev` |
| `api-key` | string | API key | None |
| `default-project` | string | Default project ID | None |
| `output-format` | string | Output format | `table` |
| `timeout` | number | Request timeout | `30` |
| `verbose` | boolean | Verbose output | `false` |

## Multiple Config Files

Use different config files for different environments:

```bash
# Use specific config
trace --config ~/.config/tracertm/prod.json project list
```
