# Python SDK Installation

The TraceRTM Python SDK provides a Pythonic interface to the TraceRTM API.

## Installation

### Using pip

```bash
pip install tracertm
```

### Using pip with specific version

```bash
pip install tracertm==2.4.0
```

### From source

```bash
git clone https://github.com/tracertm/tracertm-python.git
cd tracertm-python
pip install -e .
```

## Requirements

- Python 3.12 or higher
- pip 23.0 or higher

## Verify Installation

```python
import tracertm
print(tracertm.__version__)
```

## Quick Start

```python
from tracertm import TraceRTM

# Initialize client
client = TraceRTM(api_key="rtm_live_abc123...")

# List projects
projects = client.projects.list()
for project in projects:
    print(f"{project.id}: {project.name}")
```
