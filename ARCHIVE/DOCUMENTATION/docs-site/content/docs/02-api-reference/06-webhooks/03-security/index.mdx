# Webhook Security


## Overview

Complete API reference for Webhook Security.

## Authentication

All API requests require authentication:

```typescript
const headers = {
  'Authorization': 'Bearer YOUR_JWT_TOKEN',
  'Content-Type': 'application/json'
}
```

## Endpoints

### List All Webhook Security

```http
GET /api/v1/webhook-security
```

**Response:**
```json
{
  "data": [],
  "pagination": {
    "page": 1,
    "per_page": 20,
    "total": 100
  }
}
```

### Get Webhook Security by ID

```http
GET /api/v1/webhook-security/:id
```

### Create Webhook Security

```http
POST /api/v1/webhook-security
```

**Request Body:**
```json
{
  "name": "Example Webhook Security",
  "description": "Description here"
}
```

### Update Webhook Security

```http
PUT /api/v1/webhook-security/:id
```

### Delete Webhook Security

```http
DELETE /api/v1/webhook-security/:id
```

## Query Parameters

- `page` - Page number (default: 1)
- `per_page` - Items per page (default: 20)
- `sort` - Sort field
- `order` - Sort order (asc/desc)
- `filter` - Filter criteria

## Error Responses

### 400 Bad Request
Invalid request parameters

### 401 Unauthorized
Missing or invalid authentication token

### 404 Not Found
Resource not found

### 500 Internal Server Error
Server error
