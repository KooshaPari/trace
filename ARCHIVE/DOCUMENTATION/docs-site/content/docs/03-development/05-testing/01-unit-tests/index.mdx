## Getting Started

```bash
tracertm init
tracertm config set unit.tests enabled
```

## Usage

### Basic Usage

```typescript
import { TraceRTM } from '@tracertm/core'

const tracertm = new TraceRTM()
const result = await tracertm.unittests()
```

### Advanced Usage

```typescript
const config = {
  unit_tests: {
    enabled: true,
    options: {
      mode: 'advanced'
    }
  }
}

const tracertm = new TraceRTM(config)
```

## Configuration

```typescript
export default {
  unit_tests: {
    enabled: true,
    settings: {
      // Configuration here
    }
  }
}
```
