## Summary
- Use AKS or Container Apps; Azure Database for Postgres; Azure Cache for Redis.
- Front with Application Gateway and WAF.
- Store secrets in Key Vault; integrate with managed identities.

## Steps
```bash
az aks command invoke --name tracertm --resource-group rg --command "kubectl rollout restart deploy/api"
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
