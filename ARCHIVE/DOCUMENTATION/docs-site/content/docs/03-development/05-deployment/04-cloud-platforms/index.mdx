## Summary
- Standardize on Terraform modules per provider.
- Use managed Postgres/Redis to reduce ops overhead.
- Ensure private networking between app and data plane.

## Steps
```bash
terraform apply -var-file env/prod.tfvars
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
