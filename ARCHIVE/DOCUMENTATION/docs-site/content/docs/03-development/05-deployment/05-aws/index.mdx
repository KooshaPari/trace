title: "AWS Deployment"
description: "Deploy TraceRTM on AWS."
---

## Summary
- Use ECS or EKS for services, RDS for Postgres, ElastiCache for Redis.
- Front with ALB; enable WAF rules for basic protections.
- Store secrets in SSM Parameter Store or Secrets Manager.

## Steps
```bash
aws ecs update-service --cluster tracertm --service api --force-new-deployment
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
