description: "Deploy with pure Docker/Compose."
---

## Summary
- Single-host deployments for pilots or demos.
- Use restart policies and healthchecks.
- Persist Postgres/Redis volumes.

## Steps
```bash
docker compose -f docker-compose.prod.yml up -d api worker docs
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
