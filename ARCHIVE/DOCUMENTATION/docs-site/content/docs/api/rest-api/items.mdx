# Items

Operations for managing requirements items

## Endpoints

### List Items

<Badge variant="blue">GET</Badge> `/api/v1/items`

Retrieve a paginated list of items in a project


### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `project_id` | string (uuid) | query | ✓ | Unique identifier for the project |
| `skip` | integer | query |  | Number of items to skip (for pagination) |
| `limit` | integer | query |  | Maximum number of items to return |


### Code Examples

<Tabs items={["cURL", "JavaScript", "Python", "Go"]}>

```bash
curl -X GET "https://api.tracertm.com/api/v1/items" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

### Get Item

<Badge variant="blue">GET</Badge> `/api/v1/items/{item_id}`

Retrieve detailed information about a specific item


### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `item_id` | string (uuid) | path | ✓ | Unique identifier for the item |
