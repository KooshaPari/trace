# Installation

The TraceRTM Go SDK provides an idiomatic Go interface for the TraceRTM API with full type safety, context support, and comprehensive error handling.

## Requirements

- Go 1.21 or higher
- A TraceRTM account with an API key

---

## Installation Methods

### Using go get

The standard way to install the SDK:

```bash
go get github.com/tracertm/tracertm-go
```

### Using go mod

Add to your `go.mod` file:

```go
require github.com/tracertm/tracertm-go v1.0.0
```

Then run:

```bash
go mod download
```

### Specific Version

Install a specific version:

```bash
go get github.com/tracertm/tracertm-go@v1.2.3
```

### Latest Development

Install the latest development version:

```bash
go get github.com/tracertm/tracertm-go@latest
```

## Verify Installation

Create a test file to verify the installation:

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/tracertm/tracertm-go"
)

func main() {
    client := tracertm.NewClient(os.Getenv("TRACERTM_API_KEY"))

    ctx := context.Background()

    // Test connection
    projects, err := client.Projects.List(ctx, nil)
    if err != nil {
        log.Fatal(err)
    }

    fmt.Printf("Connected! Found %d projects\n", len(projects.Data))
}
```

Run with:

```bash
export TRACERTM_API_KEY="rtm_live_sk_..."
go run main.go
```

## Configuration

### Environment Variables

The recommended approach for production:

```bash
# Required
export TRACERTM_API_KEY="rtm_live_sk_..."

# Optional
export TRACERTM_BASE_URL="https://api.tracertm.dev/v1"
export TRACERTM_TIMEOUT="30s"
export TRACERTM_MAX_RETRIES="3"
export TRACERTM_DEBUG="false"
```

### Configuration File

Using a configuration struct:

```go
package config

import (
    "os"
    "time"
)

type TraceRTMConfig struct {
    APIKey     string
    BaseURL    string
    Timeout    time.Duration
    MaxRetries int
    Debug      bool
}

func LoadConfig() *TraceRTMConfig {
    timeout := 30 * time.Second
    if t := os.Getenv("TRACERTM_TIMEOUT"); t != "" {
        if parsed, err := time.ParseDuration(t); err == nil {
            timeout = parsed
        }
    }

    return &TraceRTMConfig{
        APIKey:     os.Getenv("TRACERTM_API_KEY"),
        BaseURL:    getEnvOrDefault("TRACERTM_BASE_URL", "https://api.tracertm.dev/v1"),
        Timeout:    timeout,
        MaxRetries: getEnvOrDefaultInt("TRACERTM_MAX_RETRIES", 3),
        Debug:      os.Getenv("TRACERTM_DEBUG") == "true",
    }
}

func getEnvOrDefault(key, defaultValue string) string {
    if value := os.Getenv(key); value != "" {
        return value
    }
    return defaultValue
}
```

### Using Viper

```go
package main

import (
    "github.com/spf13/viper"
    "github.com/tracertm/tracertm-go"
)

func main() {
    viper.SetConfigName("config")
    viper.SetConfigType("yaml")
    viper.AddConfigPath(".")
    viper.ReadInConfig()

    client := tracertm.NewClient(
        viper.GetString("tracertm.api_key"),
        tracertm.WithBaseURL(viper.GetString("tracertm.base_url")),
        tracertm.WithTimeout(viper.GetDuration("tracertm.timeout")),
    )
}
```

With `config.yaml`:

```yaml
tracertm:
  api_key: "${TRACERTM_API_KEY}"
  base_url: "https://api.tracertm.dev/v1"
  timeout: 30s
  max_retries: 3
```

## Client Initialization

### Basic Client

```go
package main

import (
    "os"
    "github.com/tracertm/tracertm-go"
)

func main() {
    client := tracertm.NewClient(os.Getenv("TRACERTM_API_KEY"))
}
```

### With Options

```go
import (
    "net/http"
    "time"
    "github.com/tracertm/tracertm-go"
)

client := tracertm.NewClient(
    os.Getenv("TRACERTM_API_KEY"),

    // Base URL
    tracertm.WithBaseURL("https://api.tracertm.dev/v1"),

    // Timeout
    tracertm.WithTimeout(30 * time.Second),

    // Retry configuration
    tracertm.WithMaxRetries(3),
    tracertm.WithRetryDelay(time.Second),

    // Custom HTTP client
    tracertm.WithHTTPClient(&http.Client{
        Timeout: 60 * time.Second,
    }),

    // Debug mode
    tracertm.WithDebug(true),

    // Custom headers
    tracertm.WithHeaders(map[string]string{
        "X-Custom-Header": "value",
    }),
)
```

### From Environment

```go
// Automatically reads from TRACERTM_API_KEY
client, err := tracertm.NewClientFromEnv()
if err != nil {
    log.Fatal(err)
}
```

### Singleton Pattern

```go
package tracertm

import (
    "os"
    "sync"

    "github.com/tracertm/tracertm-go"
)

var (
    client *tracertm.Client
    once   sync.Once
)

func GetClient() *tracertm.Client {
    once.Do(func() {
        client = tracertm.NewClient(os.Getenv("TRACERTM_API_KEY"))
    })
    return client
}
```

## Troubleshooting

### Module Not Found

```bash
go: github.com/tracertm/tracertm-go: module lookup disabled
```

**Solution**: Enable Go modules:

```bash
export GO111MODULE=on
go mod tidy
```

### Version Mismatch

```bash
go: module github.com/tracertm/tracertm-go: reading github.com/tracertm/tracertm-go: 410 Gone
```

**Solution**: Clear module cache and retry:

```bash
go clean -modcache
go mod download
```

### Import Cycle

**Solution**: Use interfaces to break cycles:

```go
// Define interface in a separate package
type ItemsClient interface {
    Get(ctx context.Context, id string) (*Item, error)
    List(ctx context.Context, projectID string, opts *ListItemsOptions) (*ItemList, error)
}
```

### Proxy Issues

If behind a corporate proxy:

```bash
export GOPROXY=https://proxy.golang.org,direct
# or for private proxy
export GOPROXY=https://your-proxy.example.com,direct
export GONOSUMDB=github.com/tracertm/*
```
