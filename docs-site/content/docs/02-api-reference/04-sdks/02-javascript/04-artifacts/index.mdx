## JavaScript SDK — Artifacts
- Recommended for applications already using JavaScript.
- Handles auth, retries, and serialization for you.

## Example
```javascript
const artifacts = await client.artifacts.list({ projectId: 'proj_123', type: 'feature' })
await client.artifacts.create({ projectId: 'proj_123', name: 'Checkout Service', type: 'code' })
```

## Tips
- Initialize the client once per process and reuse it.
- Set timeouts and retries explicitly in production.
- Capture request IDs from responses for support cases.
