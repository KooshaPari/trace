## Python SDK — Authentication
- Recommended for applications already using Python.
- Handles auth, retries, and serialization for you.

## Example
```python
from tracertm import Client
client = Client(api_key="${TRACE_API_KEY}")
client.health.check()
```

## Tips
- Initialize the client once per process and reuse it.
- Set timeouts and retries explicitly in production.
- Capture request IDs from responses for support cases.
