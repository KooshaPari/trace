# CLI Installation - Linux

Install the TraceRTM CLI on Linux using your distribution's package manager or pip.

## Ubuntu/Debian

### Using apt (if available)

```bash
curl -fsSL https://tracertm.io/install.sh | sudo bash
sudo apt-get update
sudo apt-get install tracertm-cli
```

### Using pip

```bash
# Install Python 3.12+ if needed
sudo apt-get update
sudo apt-get install python3.12 python3-pip

# Install CLI
pip3 install tracertm-cli

# Add to PATH
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

## Fedora/RHEL

### Using dnf

```bash
sudo dnf install python3.12 python3-pip
pip3 install tracertm-cli
```

## Arch Linux

### Using pacman

```bash
sudo pacman -S python python-pip
pip install tracertm-cli
```

## Verify Installation

```bash
trace --version
```

## Troubleshooting

### Command Not Found

```bash
# Add to PATH
export PATH="$HOME/.local/bin:$PATH"

# Or use python -m
python3 -m tracertm --version
```
