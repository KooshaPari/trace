## Role in the architecture
- Separate read/write workloads for scalability.
- Explicit contracts via protobuf/OpenAPI.
- Async orchestration for long-running workflows.

## Considerations
- Throughput vs. latency trade-offs.
- Failure domains and retry/backoff strategies.
- Data model ownership and boundaries.
