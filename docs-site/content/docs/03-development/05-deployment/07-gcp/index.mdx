## Summary
- Use GKE or Cloud Run; Cloud SQL for Postgres; Memorystore for Redis.
- Front with Global HTTP(S) LB and Cloud Armor.
- Use Secret Manager and Workload Identity.

## Steps
```bash
gcloud run deploy api --image gcr.io/project/api:latest --region us-central1
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
