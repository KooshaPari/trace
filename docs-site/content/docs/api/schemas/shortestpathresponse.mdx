# ShortestPathResponse

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `exists` | boolean | ✓ | Whether a path exists between the items |
| `distance` | integer | ✓ | Number of hops in the shortest path |
| `path` | string (uuid)[] | ✓ | Ordered list of item IDs in the path |
| `link_types` | string[] | ✓ | Types of links in the path |

## Example

```json
{
  "exists": true,
  "distance": 0,
  "path": [],
  "link_types": []
}
```
