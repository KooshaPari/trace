# Web UI Guide

Complete guide to using the TraceRTM web user interface.

## Getting Started

### Access the Web UI

1. Navigate to your TraceRTM instance
2. Log in with your credentials
3. Select or create a project

## Navigation

### Main Menu

- **Projects** - Manage projects
- **Items** - View and manage items
- **Links** - View traceability links
- **Search** - Search across items
- **Reports** - Generate reports

### Project View

- **Overview** - Project summary
- **Items** - All items in project
- **Links** - Link graph visualization
- **Matrix** - Traceability matrix
- **Settings** - Project configuration

## Creating Items

### Create Requirement

1. Click **"New Item"** button
2. Select **"Requirement"** view
3. Enter title and description
4. Set status and metadata
5. Click **"Create"**

### Create Test Case

1. Click **"New Item"** button
2. Select **"Test Case"** view
3. Enter test case details
4. Link to requirements
5. Click **"Create"**

## Creating Links

### Link Items

1. Open source item
2. Click **"Add Link"**
3. Select target item
4. Choose link type
5. Click **"Create Link"**

## Visualizations

### Link Graph

View traceability relationships as a graph:

- **Nodes** = Items
- **Edges** = Links
- **Colors** = Different views/types

### Traceability Matrix

View requirements vs test cases matrix:

- **Rows** = Requirements
- **Columns** = Test Cases
- **Checkmarks** = Links exist

## Search

### Basic Search

1. Click search icon
2. Enter search terms
3. View results

### Advanced Search

- Filter by view
- Filter by status
- Filter by metadata
- Sort results
