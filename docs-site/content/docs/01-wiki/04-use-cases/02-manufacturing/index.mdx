# Manufacturing

Using TraceRTM for requirements traceability in manufacturing processes.

## Quality Assurance

### Design to Production Traceability

1. **Design Specifications**
   ```python
   spec = client.items.create(
       title="Product Specification 1.0",
       view="requirements",
       description="Material, dimensions, tolerances"
   )
   ```

2. **Quality Tests**
   ```python
   test = client.items.create(
       title="Quality Test: Dimensional Check",
       view="test-cases",
       description="Verify product dimensions"
   )
   
   client.links.create(
       source_id=spec.id,
       target_id=test.id,
       link_type="verified_by"
   )
   ```

### Production Requirements

```python
# Link production requirements to design
production_req = client.items.create(
    title="Production Requirement: Batch Size",
    view="requirements"
)

client.links.create(
    source_id=production_req.id,
    target_id=spec.id,
    link_type="implements"
)
```

## Supply Chain Traceability

### Component Traceability

```python
# Track components through supply chain
component = client.items.create(
    title="Component: Motor Assembly",
    view="requirements",
    metadata={"supplier": "Supplier A", "lot": "LOT-2025-001"}
)

# Link to product
product = client.items.create(
    title="Product: Electric Vehicle",
    view="requirements"
)

client.links.create(
    source_id=product.id,
    target_id=component.id,
    link_type="contains"
)
```

## Quality Control

### Test Coverage

```python
# Track test coverage for quality requirements
quality_req = client.items.create(
    title="Quality Requirement: Durability",
    view="requirements"
)

# Link multiple tests
for test_name in ["Stress Test", "Fatigue Test", "Environmental Test"]:
    test = client.items.create(
        title=test_name,
        view="test-cases"
    )
    client.links.create(
        source_id=quality_req.id,
        target_id=test.id,
        link_type="verified_by"
    )
```

## Compliance

### Industry Standards

```python
# Link to standards
standard = client.items.create(
    title="ISO 9001:2015 Compliance",
    view="requirements",
    metadata={"standard": "ISO 9001:2015"}
)

# Link requirements to standard
requirement = client.items.create(
    title="Quality Management Requirement",
    view="requirements"
)

client.links.create(
    source_id=requirement.id,
    target_id=standard.id,
    link_type="complies_with"
)
```
