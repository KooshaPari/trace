## Getting Started

```bash
tracertm init
tracertm config set use.cases enabled
```

## Usage

### Basic Usage

```typescript
import { TraceRTM } from '@tracertm/core'

const tracertm = new TraceRTM()
const result = await tracertm.usecases()
```

### Advanced Usage

```typescript
const config = {
  use_cases: {
    enabled: true,
    options: {
      mode: 'advanced'
    }
  }
}

const tracertm = new TraceRTM(config)
```

## Configuration

export default {
