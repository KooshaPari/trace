## Getting Started

### Prerequisites

- Basic understanding of TraceRTM
- Access to a TraceRTM workspace
- Required permissions

### Initial Setup

```bash
tracertm init
tracertm config set 5.minute.tutorial enabled
```

## Core Workflows

### Workflow 1: Basic Usage

```bash
# Step 1: Initialize
tracertm 5-minute-tutorial init

# Step 2: Execute
tracertm 5-minute-tutorial run

# Step 3: Verify
tracertm 5-minute-tutorial status
```

### Workflow 2: Advanced Usage

```typescript
import { TraceRTM } from '@tracertm/core'

const tracertm = new TraceRTM()
await tracertm.5minutetutorial({
  mode: 'advanced',
  options: {
    verbose: true
  }
})
```
