## Choose Your Client

TraceRTM is available on multiple platforms. Choose the one that best fits your workflow:

<Tabs items={["Web UI", "CLI", "Desktop App"]}>
  <Tab value="Web UI">
    The web interface is the easiest way to get started. No installation required -
    just sign in at [app.tracertm.com](https://app.tracertm.com).

    **Best for:**
    - Quick access from any device
    - Visual matrix editing
    - Team collaboration
    - Report generation
  </Tab>

  <Tab value="CLI">
    The command-line interface is perfect for automation and power users.

    ```bash
    pip install tracertm
    tracertm login
    tracertm project list
    ```

    **Best for:**
    - CI/CD integration
    - Scripting and automation
    - Bulk operations
    - Git integration
  </Tab>

  <Tab value="Desktop App">
    The desktop application offers offline capabilities and native performance.

    **Best for:**
    - Offline work
    - Large datasets
    - Sensitive data (local storage)
    - Native OS integration
  </Tab>
</Tabs>
