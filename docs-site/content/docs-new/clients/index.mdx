## Web UI

The web interface is the most visual and collaborative way to use TraceRTM.

**Best for:**
- Day-to-day requirements management
- Visual traceability matrix editing
- Team collaboration
- Report generation and sharing
- Stakeholder reviews

[Go to Web UI Guide](/docs/clients/web-ui)

## CLI (Command Line Interface)

The CLI is built with Python and Typer, providing a powerful interface for
automation and scripting.

**Best for:**
- CI/CD pipeline integration
- Bulk import/export operations
- Scripting and automation
- Git workflow integration
- Power users who prefer terminals

<CLICommand
  command="tracertm item list --project my-project --format json | jq '.[] | .title'"
  description="List all item titles in a project"
  output={`"User Authentication"
"Password Reset"
"Session Management"
"Two-Factor Auth"`}
/>

[Go to CLI Guide](/docs/clients/cli)

## Desktop App

The desktop application is built with Tauri and provides native performance
with offline capabilities.

**Best for:**
- Working offline or with limited connectivity
- Large datasets that benefit from local processing
- Sensitive data that should stay on-device
- Native OS integrations (file associations, etc.)

**Supported Platforms:**
- macOS (Intel and Apple Silicon)
- Windows 10/11
- Linux (Ubuntu, Fedora, Arch)

[Go to Desktop Guide](/docs/clients/desktop)

## Getting Started

<Tabs items={["Web UI", "CLI", "Desktop"]}>
  <Tab value="Web UI">
    1. Go to [app.tracertm.com](https://app.tracertm.com)
    2. Sign in with your account
    3. Select or create a project
    4. Start managing requirements!

    [Full Web UI Guide](/docs/clients/web-ui)
  </Tab>

  <Tab value="CLI">
    ```bash
    # Install via pip
    pip install tracertm

    # Authenticate
    tracertm login

    # Verify installation
    tracertm --version
    ```

    [Full CLI Guide](/docs/clients/cli)
  </Tab>

  <Tab value="Desktop">
    1. Download from [tracertm.com/download](https://tracertm.com/download)
    2. Install for your platform
    3. Launch and sign in
    4. Choose online or offline mode

    [Full Desktop Guide](/docs/clients/desktop)
  </Tab>
</Tabs>
