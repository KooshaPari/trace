    ├── CodeReferences[] → CodeEntity (via CodeEntityID)
    ├── APIReferences[] → API endpoint handlers
    ├── ConceptLinks[] → BusinessTerm (via ConceptID)
    └── RelatedEntities[] → Other DocumentEntity nodes
      ├── Children (doc_section)
      │   ├── CodeReferences[]
      │   ├── APIReferences[]
      │   ├── ConceptLinks[]
      │   └── Children (doc_chunk)
      │       ├── CodeReferences[]
      │       ├── ConceptLinks[]
      │       └── Embedding[]

Concept Graph:
  BusinessTerm (canonical)
    ├── ConceptMention[] (document occurrences)
    ├── ConceptCluster (grouping)
    ├── RelatedTerms[]
    └── Aliases[]

Cross-Domain Links:
  ConceptLink (DocumentEntity → BusinessTerm)
  CodeReference (DocumentEntity → CodeEntity)
  APIReference (DocumentEntity → API handlers)
  TraceLink (DocumentEntity → CodeEntity, bidirectional)
