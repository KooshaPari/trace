Craft an async standup update using `$ARGUMENTS` (bullets or notes):

- **Yesterday:** Key outcomes, PRs merged, blockers resolved.
- **Today:** Focus items and intended deliverables.
- **Risks/Asks:** Dependencies, open questions, or help required.

Keep each section to ≤2 bullets. End with the next milestone or target date if known.
