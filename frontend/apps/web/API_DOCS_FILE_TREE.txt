Security Schemes
===============
Bearer Token (JWT):
  Header: Authorization: Bearer <token>
  Format: JWT
  Description: JWT Bearer token authentication

API Key:
  Header: X-API-Key: <key>
  In: header
  Description: API key authentication

Verification
===========
Run the verification script:
  ./verify-api-docs.sh

Or manually check:
  1. Packages installed: grep "swagger-ui-react\|redoc" package.json
  2. Files exist: ls -la public/specs/openapi.json
  3. Routes work: curl http://localhost:3000/api/spec
  4. Components load: Visit /api-docs in browser
