# Quick Start Guide

Get TraceRTM up and running in 10 minutes.

## Step 1: Initialize Configuration

```bash
# Initialize with SQLite (easiest for getting started)
rtm config init --database-url sqlite:///tracertm.db
```

Expected output:

```
✓ Configuration initialized successfully
Config file: ~/.tracertm/config.json
```

## Step 2: Create Your First Project

```bash
# Create a project
rtm project init "My First Project" \
  --description "Learning TraceRTM"
```

Expected output:

```
✓ Project 'My First Project' initialized successfully!
Database: sqlite:///tracertm.db
Project ID: <project-id>
```

## Step 3: Create Items

```bash
# Create an epic
rtm item create "User Management System" \
  --view FEATURE \
  --type epic \
  --description "Complete user management functionality"

# Create a feature under the epic
rtm item create "User Authentication" \
  --view FEATURE \
  --type feature \
  --parent <epic-id> \
  --description "Login and registration features"

# Create a user story
rtm item create "User can login with email" \
  --view FEATURE \
  --type story \
  --parent <feature-id> \
  --status todo
```

## Step 4: View Your Items

```bash
# List all items
rtm item list

# List only feature view items
rtm item list --view FEATURE

# Show item details
rtm item show <item-id>
```

## Step 5: Create Cross-View Links

```bash
# Create a code file
CODE_ID=$(rtm item create "auth/login.py" \
  --view CODE --type file | grep "ID:" | awk '{print $2}')

# Link feature to code
rtm link create \
  --source <feature-id> \
  --target $CODE_ID \
  --type implements
```
