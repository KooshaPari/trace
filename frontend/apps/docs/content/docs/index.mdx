# Welcome to TraceRTM

TraceRTM is a comprehensive requirements traceability and management platform designed for modern software development teams.

## Quick Links

- [Getting Started](/docs/00-getting-started) - Set up TraceRTM and create your first project

## Features

- **Multi-View System**: Manage requirements, code, tests, and APIs in unified views
- **Cross-View Traceability**: Link items across different views for complete traceability
- **Real-time Collaboration**: Work together with real-time updates via WebSocket
- **GraphQL & REST APIs**: Flexible API access for integrations
- **CLI & MCP Tools**: Command-line and Model Context Protocol tools for automation
- **Enterprise-Ready**: Built with FastAPI, PostgreSQL, and React for scale

## Getting Started

Start with our [Quick Start Guide](/docs/00-getting-started/quick-start) to get up and running in minutes.

## Architecture

TraceRTM is built with a modern, scalable architecture:

- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Frontend**: React, TanStack Router, TanStack Query
- **Real-time**: NATS, WebSocket
- **CLI**: Typer, Rich
- **MCP**: FastMCP for AI integration

Learn more in our [Architecture Overview](/docs/01-architecture).
