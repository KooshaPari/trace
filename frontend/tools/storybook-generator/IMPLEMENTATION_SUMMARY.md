## Key Features Implemented

### 1. Automatic Component Detection

- Scans `packages/ui/src/components/` and `apps/web/src/components/`
- Filters by pattern (`.tsx` files, excludes tests/stories)
- Handles forwardRef and function component patterns
- Supports both named and default exports

### 2. CVA Variant Detection

- Parses class-variance-authority configuration
- Extracts variant names, options, and default values
- Generates individual stories for each variant option
- Creates showcase story with all variants

### 3. TypeScript Props Extraction

- Uses ts-morph for AST analysis
- Extracts props from interfaces and type aliases
- Generates appropriate controls (select, boolean, text, number)
- Preserves type safety

### 4. Design System Integration

- Loads Figma URLs from YAML metadata
- Adds design parameters to stories
- Includes trace IDs for requirements linking
- Auto-creates example metadata file

### 5. CSF 3.0 Story Generation

- Modern Storybook format
- Auto-docs enabled by default
- Type-safe StoryObj usage
- Consistent story structure

## Technical Architecture

### Component Analysis Flow

```
Component File (.tsx)
    ↓
ts-morph Parser
    ↓
AST Analysis
    ├─→ Component Name
    ├─→ Export Type
    ├─→ CVA Variants
    └─→ Props Interface
    ↓
ComponentInfo Object
```

### Story Generation Flow

```
ComponentInfo + DesignMetadata
    ↓
StoryTemplateConfig
    ↓
Template Generator
    ├─→ Meta Configuration
    ├─→ ArgTypes
    ├─→ Parameters (Figma, Trace)
    ├─→ Variant Stories
    └─→ Showcase Story
    ↓
Generated .stories.tsx File
```

For a component with CVA variants:

**Input (Button.tsx):**

```typescript
const buttonVariants = cva('base', {
  variants: {
    variant: { default: '...', destructive: '...', outline: '...' },
    size: { default: '...', sm: '...', lg: '...' },
  },
  defaultVariants: { variant: 'default', size: 'default' },

```

**Output (Button.stories.tsx):**

```typescript
// Auto-generated by TraceRTM Storybook Generator
import type { Meta, StoryObj } from '@storybook/react-vite'
import { Button } from '@tracertm/ui'

const meta: Meta<typeof Button> = {
  title: 'Components/Button',
  component: Button,
  tags: ['autodocs'],
  argTypes: {
    variant: {
      control: 'select',
      options: ['default', 'destructive', 'outline'],
    },
    size: {
      control: 'select',
      options: ['default', 'sm', 'lg'],
    }
  },
  parameters: {
    design: {
      type: 'figma',
      url: 'https://www.figma.com/file/xxx/Design-System?node-id=123',
    },
    trace: {
      componentId: 'comp-button-001',
      storyId: 'story-button-001',
    }
  },
}

export default meta
type Story = StoryObj<typeof meta>

export const Default: Story = {
  args: { children: 'Default', variant: 'default' },
}

export const Destructive: Story = {
  args: { children: 'Destructive', variant: 'destructive' },
}

export const Outline: Story = {
  args: { children: 'Outline', variant: 'outline' },
}

export const Sm: Story = {
  args: { children: 'Sm', size: 'sm' },
}

export const Lg: Story = {
  args: { children: 'Lg', size: 'lg' },
}

export const AllVariants: Story = {
  render: () => (
    <div className="flex flex-wrap gap-4">
      <Button variant="default">Default</Button>
      <Button variant="destructive">Destructive</Button>
      <Button variant="outline">Outline</Button>
    </div>
  ),
}
```
