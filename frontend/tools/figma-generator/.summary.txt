TAILWIND → FIGMA MAPPINGS
================================================================================

Layout:
  flex               → layoutMode: 'HORIZONTAL'
  flex-col           → layoutMode: 'VERTICAL'
  gap-{n}            → itemSpacing: {n*4}px

Colors:
  bg-{color}-{shade} → fills: [{color}]
  text-{color}       → typography.color

Spacing:
  p-{n}              → padding (all sides)
  px-{n}, py-{n}     → padding (axis)
  pt/r/b/l-{n}       → padding (side)

Typography:
  text-{size}        → fontSize
  font-{weight}      → fontWeight

Effects:
  rounded-{size}     → cornerRadius
  shadow-{size}      → dropShadow effect

USAGE EXAMPLE
================================================================================

# Setup
cp .env.figma.example .env.local
# Add FIGMA_ACCESS_TOKEN and FIGMA_FILE_KEY

# Pull design tokens
bun run figma:pull

# Export to TypeScript
bun run figma:export-tokens src/design-tokens.ts

# Generate Figma plugin
bun run figma:export

# Import to Figma
Plugins → Development → Import plugin from manifest
Select .figma-output/manifest.json

# Two-way sync
bun run figma:sync
