## Buffer & Contingency

### Built-in Buffers

- **Sprint 8:** Intensive sprint (50-60 hours) - can extend to 3 weeks if needed
- **Between Phases:** 1 week buffer between Sprint 8 and 9 for MVP stabilization
- **Each Sprint:** 20% time buffer for unexpected issues

### Contingency Plans

**If Behind Schedule:**
1. Reduce scope of Sprint 8 (defer some Epic 7/8 stories to Phase 2)
2. Extend sprint length from 2 weeks to 3 weeks
3. Defer Phase 2 entirely (ship MVP only)

**If Ahead of Schedule:**
1. Add more test coverage
2. Improve documentation
3. Start Phase 2 early
