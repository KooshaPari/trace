# Go SDK - Workflows

Manage item status workflows and transitions using the Go SDK.

## Transition Item Status

```go
result, err := client.Items.Transition("item_abc123", tracertm.TransitionRequest{
    ToStatus: "active",
    Comment:  "Ready for review",
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Transitioned from %s to %s\n", 
    result.PreviousStatus, result.NewStatus)
```

## Get Workflow

```go
workflow, err := client.Workflows.Get("proj_123")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Workflow: %s\n", workflow.Name)
fmt.Printf("States: %v\n", workflow.States)
```
