# Go SDK Installation

The TraceRTM Go SDK provides a Go interface to the TraceRTM API.

## Installation

```bash
go get github.com/tracertm/tracertm-go
```

## Requirements

- Go 1.21 or higher

## Quick Start

```go
package main

import (
    "fmt"
    "github.com/tracertm/tracertm-go"
)

func main() {
    client := tracertm.NewClient(tracertm.WithAPIKey("rtm_live_abc123..."))
    
    projects, err := client.Projects.List()
    if err != nil {
        panic(err)
    }
    
    for _, project := range projects {
        fmt.Printf("%s: %s\n", project.ID, project.Name)
    }
}
```
