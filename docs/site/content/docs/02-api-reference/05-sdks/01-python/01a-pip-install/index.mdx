# Pip Installation

Install the TraceRTM Python SDK using pip, Python's standard package manager.

## Basic Installation

```bash
pip install tracertm
```

This installs the latest stable version of the SDK.

---

## Version-Specific Installation

### Install Specific Version

```bash
# Install exact version
pip install tracertm==1.2.3

# Install minimum version
pip install "tracertm>=1.2.0"

# Install version range
pip install "tracertm>=1.2.0,<2.0.0"
```

### Install Pre-release

```bash
# Install latest pre-release
pip install --pre tracertm

# Install specific beta
pip install tracertm==2.0.0b1
```

---

## Installation with Extras

The SDK provides optional extras for additional functionality:

```bash
# Install with async support (includes aiohttp)
pip install "tracertm[async]"

# Install with CLI tools
pip install "tracertm[cli]"

# Install with development tools
pip install "tracertm[dev]"

# Install with all extras
pip install "tracertm[all]"

# Combine extras
pip install "tracertm[async,cli]"
```

### Available Extras

| Extra | Description | Dependencies |
|-------|-------------|--------------|
| `async` | Async client support | aiohttp, aiofiles |
| `cli` | Command-line interface | click, rich |
| `dev` | Development tools | pytest, mypy, black |
| `docs` | Documentation generation | sphinx, furo |
| `all` | All optional dependencies | Everything above |

## Virtual Environment Installation

### Using venv (Recommended)

```bash
# Create virtual environment
python -m venv .venv

# Activate (Linux/macOS)
source .venv/bin/activate

# Activate (Windows)
.venv\Scripts\activate

# Install SDK
pip install tracertm
```

### Using virtualenv

```bash
# Install virtualenv
pip install virtualenv

# Create environment
virtualenv .venv

# Activate and install
source .venv/bin/activate
pip install tracertm
```

## Requirements File

### Basic requirements.txt

```txt
tracertm>=1.0.0
```

### Pinned Dependencies

```txt
# requirements.txt
tracertm==1.2.3
requests==2.31.0
pydantic==2.5.0
```

### Install from Requirements

```bash
pip install -r requirements.txt
```

### Generate Requirements

```bash
# Freeze current environment
pip freeze > requirements.txt

# Export only top-level packages
pip list --format=freeze > requirements.txt
```

## Upgrade and Uninstall

### Upgrade SDK

```bash
# Upgrade to latest version
pip install --upgrade tracertm

# Upgrade to specific version
pip install --upgrade tracertm==1.3.0

# Check current version
pip show tracertm
```

### Uninstall SDK

```bash
pip uninstall tracertm
```

## Troubleshooting

### Permission Errors

```bash
# Use --user flag for user-level installation
pip install --user tracertm

# Or use sudo (not recommended)
sudo pip install tracertm
```

### SSL Certificate Errors

```bash
# Trust PyPI certificates
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org tracertm
```

### Proxy Configuration

```bash
# Set proxy
pip install --proxy http://proxy.example.com:8080 tracertm

# Or use environment variable
export HTTP_PROXY=http://proxy.example.com:8080
export HTTPS_PROXY=http://proxy.example.com:8080
pip install tracertm
```

### Offline Installation

```bash
# Download packages
pip download tracertm -d ./packages

# Install from downloaded packages
pip install --no-index --find-links=./packages tracertm
```
