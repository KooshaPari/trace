# CLI Installation - macOS

Install the TraceRTM CLI on macOS using Homebrew (recommended) or pip.

## Method 1: Homebrew (Recommended)

### Install Homebrew

If you don't have Homebrew installed:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Install TraceRTM CLI

```bash
brew install tracertm-cli
```

### Upgrade

```bash
brew upgrade tracertm-cli
```

### Uninstall

```bash
brew uninstall tracertm-cli
```

## Method 2: pip

### Install Python 3.12+

```bash
# Using Homebrew
brew install python@3.12

# Or download from python.org
```

### Install CLI

```bash
pip3 install tracertm-cli
```

### Add to PATH

If `trace` command is not found:

```bash
# Add to ~/.zshrc or ~/.bash_profile
echo 'export PATH="$HOME/Library/Python/3.12/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## Verify Installation

```bash
trace --version
```

Expected output:
```
tracertm-cli version 2.4.0
```

## Troubleshooting

### Command Not Found

```bash
# Check installation
pip3 show tracertm-cli

# Use python -m instead
python3 -m tracertm --version
```

### Permission Errors

```bash
# Use --user flag
pip3 install --user tracertm-cli
```
