# Plugin Development

Create custom CLI plugins to extend TraceRTM functionality.

## Plugin Structure

A plugin is a Python package with this structure:

```
my-plugin/
  __init__.py
  plugin.py
  setup.py
```

## Basic Plugin

```python
# plugin.py
from tracertm.plugin import Plugin, Command

class MyPlugin(Plugin):
    name = "my-plugin"
    version = "1.0.0"
    
    def register_commands(self, cli):
        @cli.command()
        def my_command():
            """My custom command"""
            print("Hello from plugin!")
```

## Advanced Plugin

```python
from tracertm.plugin import Plugin, Command
from tracertm import TraceRTM

class AdvancedPlugin(Plugin):
    name = "advanced-plugin"
    version = "1.0.0"
    
    def register_commands(self, cli):
        @cli.command()
        @cli.option('--project-id', required=True)
        def analyze_project(project_id):
            """Analyze project traceability"""
            client = TraceRTM()
            # Custom analysis logic
            pass
```

## Packaging

Create `setup.py`:

```python
from setuptools import setup

setup(
    name="tracertm-plugin-example",
    version="1.0.0",
    py_modules=["plugin"],
    entry_points={
        "tracertm.plugins": [
            "example = plugin:MyPlugin"
        ]
    }
)
```

## Distribution

### Local Installation

```bash
pip install -e /path/to/plugin
```

### PyPI

```bash
python setup.py sdist bdist_wheel
twine upload dist/*
```
