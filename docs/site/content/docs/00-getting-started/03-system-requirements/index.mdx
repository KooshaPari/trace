# System Requirements

These requirements cover local development, docs, CLI, and minimal backend services for TraceRTM.

## Overview

- **Hardware**: 4+ cores, 16GB RAM recommended for running API + worker + docs locally.
- **OS**: macOS, Linux (x86_64/arm64), Windows (WSL recommended).
- **Network**: Access to package registries and containers; optional outbound ports for Postgres/Redis/NATS if remote.

## Runtime prerequisites

- **Docker**: For Postgres, Redis, NATS; optional but recommended.
- **Node**: v18+ (docs/frontend), v20+ preferred.
- **Go**: 1.21+ for backend.
- **Python**: 3.10+ for CLI/agent scripts.

## Local service defaults

- Postgres: `localhost:5432`
- Redis: `localhost:6379`
- NATS: `localhost:4222`

## Performance tips

- Allocate Docker resources (CPU/memory) to avoid contention.
- Use SSD-backed volumes for databases.
- Run only needed services (compose profiles) to keep resource use low.
