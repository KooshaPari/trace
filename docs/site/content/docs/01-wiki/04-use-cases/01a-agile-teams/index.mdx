# Agile Teams

Using TraceRTM to maintain traceability in agile development workflows.

## Agile Traceability

### User Stories to Acceptance Criteria

1. **Create User Story**
   ```python
   story = client.items.create(
       title="As a user, I want to log in",
       view="requirements",
       status="backlog"
   )
   ```

2. **Create Acceptance Criteria**
   ```python
   criteria = client.items.create(
       title="User can log in with email/password",
       view="requirements",
       parent_id=story.id
   )
   ```

3. **Link to Test Cases**
   ```python
   test = client.items.create(
       title="Test login with valid credentials",
       view="test-cases"
   )
   client.links.create(
       source_id=criteria.id,
       target_id=test.id,
       link_type="verified_by"
   )
   ```

## Sprint Management

### Organize by Sprint

```python
# Tag items with sprint
item = client.items.create(
    title="User Story",
    metadata={"sprint": "Sprint 5", "epic": "Authentication"}
)
```

### Track Sprint Progress

```python
# Get items by sprint
sprint_items = client.items.list(
    project_id=project_id,
    metadata_filter={"sprint": "Sprint 5"}
)
```

## Epic to Story Traceability

### Link Stories to Epics

```python
# Create epic
epic = client.items.create(
    title="User Authentication Epic",
    view="requirements"
)

# Link stories to epic
story = client.items.create(
    title="Login Story",
    metadata={"epic": epic.id}
)
```
