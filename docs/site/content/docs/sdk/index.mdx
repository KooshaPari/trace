# SDK Documentation

Official Software Development Kits (SDKs) for integrating with TraceRTM.

## Available SDKs

### Python SDK

The official Python SDK for TraceRTM.

```python
from tracertm import TraceRTM

client = TraceRTM(api_key="rtm_live_abc123...")
project = client.projects.create(name="My Project")
```

[Python SDK Documentation →](/docs/api-reference/sdks/python/installation)

### JavaScript SDK

The official JavaScript/TypeScript SDK for TraceRTM.

```javascript
import { TraceRTM } from '@tracertm/sdk';

const client = new TraceRTM({ apiKey: 'rtm_live_abc123...' });
const project = await client.projects.create({ name: 'My Project' });
```

[JavaScript SDK Documentation →](/docs/api-reference/sdks/javascript/installation)

### Go SDK

The official Go SDK for TraceRTM.

```go
import "github.com/tracertm/go-sdk"

client := tracertm.NewClient("rtm_live_abc123...")
project, err := client.Projects.Create(&tracertm.CreateProjectRequest{
    Name: "My Project",
})
```

[Go SDK Documentation →](/docs/api-reference/sdks/go/installation)

## Quick Start

### Python

```bash
pip install tracertm
```

```python
from tracertm import TraceRTM

client = TraceRTM(api_key="rtm_live_abc123...")
items = client.items.list(project_id="proj_123")
```

### JavaScript

```bash
npm install @tracertm/sdk
```

```javascript
import { TraceRTM } from '@tracertm/sdk';

const client = new TraceRTM({ apiKey: 'rtm_live_abc123...' });
const items = await client.items.list({ projectId: 'proj_123' });
```

### Go

```bash
go get github.com/tracertm/go-sdk
```

```go
import "github.com/tracertm/go-sdk"

client := tracertm.NewClient("rtm_live_abc123...")
items, err := client.Items.List("proj_123")
```

## Features

All SDKs provide:

- ✅ **Full API Coverage** - All endpoints supported
- ✅ **Type Safety** - TypeScript/Go types, Python type hints
- ✅ **Error Handling** - Comprehensive error handling
- ✅ **Async Support** - Async/await in Python and JavaScript
- ✅ **Authentication** - API keys, OAuth, JWT support
- ✅ **Pagination** - Built-in pagination helpers
- ✅ **Retry Logic** - Automatic retries for transient errors
