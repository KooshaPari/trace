## Role in the architecture
- Ingress through API, persisted to Postgres, indexed in search.
- Events emitted for downstream consumers (search, analytics).
- Backpressure managed via queues with retry policies.

## Considerations
- Throughput vs. latency trade-offs.
- Failure domains and retry/backoff strategies.
- Data model ownership and boundaries.
