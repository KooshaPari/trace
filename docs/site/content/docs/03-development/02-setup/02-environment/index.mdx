---

## Steps
- Install required runtimes: Node, Go, Python, Docker.
- Copy `.env.example` to `.env` and set API keys.
- Start dependencies with Docker Compose.

```bash
cp .env.example .env
docker compose up -d postgres redis nats
```

## Tips
- Keep separate env files per project or profile.
- Use direnv to auto-load env vars when entering the repo.
