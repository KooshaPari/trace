title: "On-Prem Deployment"
description: "Run TraceRTM on-prem."
---

## Summary
- Deploy via Kubernetes or Docker Compose.
- Provide HA Postgres and Redis; configure backups.
- Harden ingress with mTLS and network policies.

## Steps
```bash
docker compose -f docker-compose.prod.yml up -d
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
