## Summary
- Ship logs to your SIEM with structured JSON.
- Expose Prometheus metrics; set SLO alerts for latency and errors.
- Trace key workflows (create/link/search) end-to-end.

## Steps
```bash
kubectl apply -f monitoring/prometheus.yaml
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
