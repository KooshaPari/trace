## Summary
- Immutable images, declarative manifests.
- Config injected via environment/secrets.
- Blue/green or rolling strategies by default.

## Steps
```bash
kubectl rollout restart deploy/api
```

## Operational checks
- Health endpoints return 200 for API and worker pods.
- Background queues drain within SLO after deploy.
- Logs and metrics flowing to your observability stack.
