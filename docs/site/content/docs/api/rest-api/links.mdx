# Links

Operations for managing relationships between items

## Endpoints

### List Links

<Badge variant="blue">GET</Badge> `/api/v1/links`

Retrieve a paginated list of links between items in a project


### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `project_id` | string (uuid) | query | ✓ | Unique identifier for the project |
| `skip` | integer | query |  |  |
| `limit` | integer | query |  |  |
