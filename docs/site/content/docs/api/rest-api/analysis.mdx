# Analysis

Advanced analysis operations including impact analysis, cycle detection, and path finding

## Endpoints

### Impact Analysis

<Badge variant="blue">GET</Badge> `/api/v1/analysis/impact/{item_id}`

Analyze the impact of changes to a specific item, including all downstream dependencies


### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `item_id` | string (uuid) | path | ✓ | Unique identifier for the item to analyze |
| `project_id` | string (uuid) | query | ✓ | Unique identifier for the project |


### Code Examples

<Tabs items={["cURL", "JavaScript", "Python", "Go"]}>

<Tab value="cURL">

```bash
curl -X GET "https://api.tracertm.com/api/v1/analysis/impact/{item_id}" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

</Tab>

<Tab value="JavaScript">

```javascript

  }
```

### Find Shortest Path

<Badge variant="blue">GET</Badge> `/api/v1/analysis/shortest-path`

Find the shortest dependency path between two items


### Parameters

| Name | Type | Location | Required | Description |
|------|------|----------|----------|-------------|
| `project_id` | string (uuid) | query | ✓ | Unique identifier for the project |
| `source_id` | string (uuid) | query | ✓ | Starting item ID |
| `target_id` | string (uuid) | query | ✓ | Target item ID |
