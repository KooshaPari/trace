# Health

Health check and system status endpoints

## Endpoints

### Health Check

<Badge variant="blue">GET</Badge> `/health`

Check the health status of the API service


### Code Examples

<Tabs items={["cURL", "JavaScript", "Python", "Go"]}>

<Tab value="cURL">

```bash
curl -X GET "https://api.tracertm.com/health" \
  -H "Content-Type: application/json"
```

</Tab>

<Tab value="JavaScript">

```javascript
const response = await fetch('https://api.tracertm.com/health', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
console.log(data);
```

</Tab>

<Tab value="Python">

```python
import requests

response = requests.get(
    "https://api.tracertm.com/health",
)
data = response.json()
print(data)
```
