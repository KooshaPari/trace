# CycleDetectionResponse

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `has_cycles` | boolean | ✓ | Whether cycles were detected |
| `total_cycles` | integer | ✓ | Number of cycles found |
| `severity` | enum: `none`, `low`, `medium`, `high`, `critical` | ✓ | Severity level of detected cycles |
| `affected_items` | string (uuid)[] | ✓ | Items involved in cycles |

## Example

```json
{
  "has_cycles": true,
  "total_cycles": 0,
  "severity": "example",
  "affected_items": []
}
```
