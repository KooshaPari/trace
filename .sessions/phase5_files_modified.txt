## Key Patterns Applied

### Pattern 1: Remove db_session Fixture
```bash
sed -i '' 's/async def test_\([^(]*\)(db_session: AsyncSession):/async def test_\1():/' file.py
```

### Pattern 2: Mock Session Initialization
```bash
sed -i '' 's/service = Service(db_session)/mock_session = MagicMock(spec=AsyncSession)\n    service = Service(mock_session)/' file.py
```

### Pattern 3: Fix Patch Targets
```bash
sed -i '' 's/with patch.object(db_session, /with patch.object(mock_session, /' file.py
```

### Pattern 4: Async Method Calls
```python
# Before
result = await service.detect_cycles(project_id="proj1")

# After
result = await service.detect_cycles_async(project_id="proj1")
```

## Test Execution Environment

### Critical Discovery
Tests were running with wrong pytest version:
- System: pytest 8.4.2 (miniforge3)
- Project: pytest 9.0.1 (uv environment)

### Solution
All tests must run with: `uv run pytest`
This ensures pytest-asyncio auto mode works correctly.
